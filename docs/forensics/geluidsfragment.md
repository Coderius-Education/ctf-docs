# Forensics - Geluidsfragment

> **[Open deze challenge op ctf.hackchallenges.nl](https://ctf.hackchallenges.nl/)** - zoek naar *"Geluidsfragment"* onder Forensics.

**Moeilijkheid:** ⭐⭐⭐ Moeilijk

## Beschrijving
Je krijgt een geluidsfragment dat onbegrijpelijk klinkt. Hoe maak je het begrijpelijk?

## Hints

:::tip Hint 1
Het geluid klinkt misschien vreemd omdat het is **omgekeerd**, **versneld**, **vertraagd**, of op een andere manier is gemanipuleerd.
:::

:::tip Hint 2
Gebruik een audio-editor zoals [Audacity](https://www.audacityteam.org/) (gratis) om het geluidsfragment te bewerken. Probeer het om te keren (reverse), de snelheid aan te passen, of het spectrogram te bekijken.
:::

:::tip Hint 3
Bekijk ook het **spectrogram** van het geluid. Soms is er een boodschap visueel verborgen in de frequenties van het geluid.
:::

## Wat je leert
- Hoe audio-bestanden gemanipuleerd kunnen worden
- Hoe je Audacity of vergelijkbare tools kunt gebruiken
- Wat een spectrogram is en hoe je het kunt lezen
