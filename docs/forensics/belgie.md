# Forensics - België

> **[Open deze challenge op ctf.hackchallenges.nl](https://ctf.hackchallenges.nl/)** - zoek naar *"België"* onder Forensics.

**Moeilijkheid:** ⭐ Makkelijk

## Onderwerp
**IP-adres**

## Beschrijving
Je krijgt een IP-adres. Hoe vind je de locatie die bij dit IP-adres hoort?

## Hints

:::tip Hint 1
Er bestaan online tools waarmee je de **geolocatie** van een IP-adres kunt opzoeken.
:::

:::tip Hint 2
Zoek op Google naar "IP address geolocation" of "IP lookup". Websites zoals [iplocation.net](https://www.iplocation.net/) of [whatismyipaddress.com](https://whatismyipaddress.com/) kunnen helpen.
:::

:::tip Hint 3
Vul het IP-adres in bij een van deze diensten. De locatie die je vindt is het antwoord op de challenge.
:::

## Wat je leert
- Wat een IP-adres is
- Hoe je de geografische locatie van een IP-adres kunt vinden
- Het verschil tussen IPv4 en IPv6
