# Forensics - Voor altijd

> **[Open deze challenge op ctf.hackchallenges.nl](https://ctf.hackchallenges.nl/)** - zoek naar *"Voor altijd"* onder Forensics.

**Moeilijkheid:** ‚≠ê Makkelijk

## Onderwerp
**Wayback Machine**

## Beschrijving
Sommige informatie op het internet verdwijnt nooit helemaal. Zelfs als een website offline gaat, kan de inhoud ergens bewaard zijn gebleven.

## Hints

:::tip Hint 1
Ga naar de **Wayback Machine**. Dit is een dienst die snapshots van websites opslaat over tijd.
:::

:::tip Hint 2
De Wayback Machine is te vinden op [web.archive.org](https://web.archive.org). Zoek daar naar de URL die in de challenge wordt gegeven.
:::

:::tip Hint 3
Bekijk de verschillende snapshots (tijdstippen) die beschikbaar zijn. De flag kan verborgen zijn in een oudere versie van de pagina.
:::

## Wat je leert
- Hoe de Wayback Machine werkt
- Dat informatie op het internet zelden echt verdwijnt
- Hoe je historische versies van websites kunt bekijken
