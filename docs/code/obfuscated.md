# Code - Obfuscated

> **[Open deze challenge op ctf.hackchallenges.nl](https://ctf.hackchallenges.nl/)** - zoek naar *"Obfuscated"* onder Code.

**Moeilijkheid:** ⭐⭐ Gemiddeld

## Onderwerp
**Code obfuscation**

## Beschrijving
De code is moeilijk te lezen omdat deze is **geobfusceerd** (onleesbaar gemaakt). Maar kun je de code wel uitvoeren?

## Wat je leert
- Wat code obfuscation is en waarvoor het wordt gebruikt
- Dat geobfusceerde code nog steeds uitvoerbaar is
- Hoe je geobfusceerde code kunt deobfusceren

## Hints

<details>
<summary>Hint 1</summary>

**Obfuscation** betekent dat code opzettelijk onleesbaar is gemaakt, maar de functionaliteit blijft hetzelfde. Je hoeft de code niet te begrijpen om hem uit te voeren.

</details>

<details>
<summary>Hint 2</summary>

Kopieer de code en voer hem uit in de juiste omgeving. Als het JavaScript is, plak het in de **browserconsole** (F12 > Console).

</details>

<details>
<summary>Hint 3</summary>

Je kunt ook proberen de code te **deobfusceren** met online tools zoals:
- [deobfuscate.io](https://deobfuscate.io/)
- [beautifier.io](https://beautifier.io/)

</details>
