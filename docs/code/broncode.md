# Code - Broncode

> **[Open deze challenge op ctf.hackchallenges.nl](https://ctf.hackchallenges.nl/)** - zoek naar *"Broncode"* onder Code.

**Moeilijkheid:** ‚≠ê Makkelijk

## Beschrijving
De flag is verborgen in de broncode van de pagina. Zoek goed!

## Hints

:::tip Hint 1
Klik met de rechtermuisknop op de pagina en kies **"Paginabron bekijken"** of druk op **Ctrl+U**.
:::

:::tip Hint 2
Gebruik **Ctrl+F** om te zoeken in de broncode. Zoek naar termen als "flag", "CTF", of het flag-formaat dat bij deze CTF wordt gebruikt.
:::

:::tip Hint 3
De flag kan verborgen zijn in:
- HTML-commentaren (`<!-- ... -->`)
- Verborgen elementen (`display: none` of `hidden`)
- Data-attributen
- JavaScript-variabelen
:::

## Wat je leert
- Hoe je de broncode van een webpagina kunt bekijken
- Hoe informatie verborgen kan worden in HTML
- Het verschil tussen wat je ziet en wat er werkelijk in de code staat
